CSS POLECENIA

color, background
display:block ustawia bloki jeden POD drugim. Domyślnie display:block; ustawione mają znaczniki: main, section, article, p, div.
display:inline ustawia bloki jeden OBOK drugiego. Domyślnie display inline ma znacznik a.
display:inline-block w opcji ułożenia inline niestety nie da się robić wielu funkcji, np. width, height, które można wykonywać w ułożeniu blokowym. Po to jest inline-block aby móc używać funkcji blockowych dla divów ułożonych inlinowo.
width i height ustawia nam stałą szerokość znacznika, diva.
min-width, min-height oznacza że minimum musi być dana wartość ale nie ma określenia max dlatego treść będzie wypełniała diva ile chce. Jak wyświetlanie będzie poniżej ustawionej wartości min wtedy pojawi się scroll.
max-width, max height oznacza maksymalną wielkość diva. jeśli treści jest więcej, wtedy ona wychodzi poza diva.
overflow jak treść wychodzi poza rozmiar boxa, diva, wtedy stosujemy overflow. Może być wartość hidden wtedy treść jest chowana, może być wartość auto wtedy treść jest scrollowana w ramach rozmiaru diva.
